# 005. UX 플로우

## 첫 실행 (온보딩)
1. "아이고" 앱 소개 — "내 아이 것은 내가 고른다"
2. 아이 정보 입력 (선택사항, 건너뛰기 가능)
   - 아이 별명 (예: "쪼꼬미", "우리 공주") — 홈 화면, 알림 문구에 사용
   - 생년월일 — 월령 자동 계산, 기념일 알림, 구간별 큐레이션
   - 성별 (남아 / 여아 / 선택안함) — 맞춤 상품 추천, 선물 큐레이션에 반영
3. 쿠팡 공유 버튼 안내 (지금이야와 동일 인터랙티브 튜토리얼)
4. 완료 → 메인 화면

## 상품 등록 (2가지)
1. **Share Intent**: 쿠팡 앱 → 공유하기 → 아이고 → 카테고리 자동분류 → 목표가 입력 → 저장
2. **수동**: 아이고 앱 → + 버튼 → URL 붙여넣기 → 카테고리 자동분류 → 목표가 입력 → 저장

## 카테고리 자동 분류 로직
- 상품명 키워드 매칭:
  - 기저귀: "기저귀", "팬티형", "밴드형", "하기스", "보솜이", "팸퍼스"
  - 분유: "분유", "액상분유", "산양분유", "남양", "매일", "앱솔루트"
  - 물티슈: "물티슈", "캡형", "도톰한", "순한"
  - 의류: "바디슈트", "우주복", "내복", "양말", "턱받이"
  - 장난감: "장난감", "블록", "놀이", "러닝", "피셔프라이스"
  - 유모차/카시트: "유모차", "카시트", "바운서", "하이체어"
  - 수유용품: "젖병", "노리개", "수유쿠션", "유축기"
  - 기타: 분류 실패 시
- 수동 변경 가능 (드롭다운)

## 알림 → 구매 플로우
1. 가격 체크 봇 (GitHub Actions, 3회/일)
2. 목표가 이하 감지 → 푸시 알림
3. 알림 탭 → 아이고 앱 상세화면
4. "지금 구매하기" → 파트너스 딥링크로 쿠팡 이동
5. 구매 → 수수료 수익

## 홈 화면 섹션 구조
1. **상단 헤더**: 아이 별명 + 월령 + 태어난 지 N일째 + 공유 버튼
   - 예: "쪼꼬미 · 8개월 · 태어난 지 245일째 🌱"
   - 별명/생년월일 미설정 시: "아이고" + "아이 정보를 입력해보세요" 배너
2. **기념일 배너** (D-7부터 자동 노출, 해당 기간에만)
   - 예: "쪼꼬미가 태어난 지 100일이에요! 🎉 백일 선물로 이런 건 어떨까요?"
   - 탭 → 기념일 선물 큐레이션 화면 (나이 + 성별 기반 쿠팡 키워드 검색 결과)
   - 하단에 부모 자축 선물 섹션 함께 노출
     - 예: "100일 동안 수고하셨어요! 오늘은 나를 위한 선물도 어떨까요?"
     - 카테고리: 뷰티, 건강식품, 의류, 홈케어
3. **계정 연동 배너** (미연동 시만 표시, "나중에" 누르면 숨김)
4. **시즌 컬렉션** (해당 시즌 D-day 기간에만, 3세 이상)
   - 예: "8세 남아 어린이날 인기 선물 TOP 10" (D-30부터)
   - 쿠팡 파트너스 키워드 검색 기반 자동 큐레이션 (성별 반영)
   - 어버이날 (5/8 D-7), 부부의 날 (5/21 D-7): 부모 선물 섹션 노출
5. **섹션1 — "○개월 맘들의 픽"**: 가로 스크롤 카드
   - 사용자 아이 월령 + 성별 → 해당 구간 자동 매칭
   - 초기: 쿠팡 파트너스 API 베스트셀러
   - 성장 후: shared_products trackerCount + ageGroup
6. **섹션2 — "이번 주 핫딜"**: 가로 스크롤 카드
   - 최근 가격 하락률 높은 상품
7. **카테고리 필터 칩** (전체/기저귀/분유/물티슈/의류/장난감/…)
8. **섹션3 — "내가 추적 중인 상품"**: 등록 상품 리스트 (카테고리 필터 적용)
9. **섹션4 — "재구매 알림 임박"**: D-day 배지 상품 (D-3 이내만 표시)
10. **FAB(+)**: URL 붙여넣기 모달

## 재구매 알림 플로우 (소모품)
1. 소모품 카테고리 상품 등록 시 → "재구매 알림 받으실래요?" 토글
2. 카테고리별 기본 주기 제안: 기저귀 14일, 분유 21일, 물티슈 30일
3. 사용자 커스텀 가능 (7~90일)
4. 주기 도래 시 푸시: "기저귀 재구매할 시간이에요! 현재 최저가 확인하기"
5. 알림 탭 → 해당 상품 상세 → 최신 가격 확인 → 구매

### "구매했어요" 버튼
- 쿠팡 앱과 데이터 연동 불가 → 사용자가 직접 구매일 기록
- 상품 상세 화면 "지금 구매하기" 옆에 "구매했어요" 버튼 배치
- 탭 시 → `lastPurchasedAt` 갱신 → 재구매 주기 리셋
- 구매일 미기록 시 → 추적 등록일(`createdAt`) 기준 주기 계산 (fallback)
- 예: "기저귀 등록한 지 14일 됐어요. 슬슬 필요할 때 아닌가요?"

## "사주세요" 공유 플로우
1. 상품 상세 화면 하단 CTA 영역에 "사주세요" 버튼 배치
   - [지금 구매하기] [사주세요 💝] [구매했어요]
2. "사주세요" 탭 → 공유 문구 선택 바텀시트
   - "쪼꼬미가 이거 갖고 싶대요! 눌러보세요 👇"
   - "할머니~ 이거 사주세요! 💝"
   - "아빠~ 쪼꼬미 생일선물이에요!"
   - 직접 입력도 가능
3. 문구 선택 → 파트너스 딥링크(subId: `share_buyme`) 생성
4. 카카오톡 / 문자 / 기타 공유 시트 열림
5. 받는 사람이 링크 클릭 → 쿠팡 구매 → 24시간 내 파트너스 수익 발생

## 앱 공유하기
- 홈 화면 헤더 공유 버튼 또는 설정 화면 "앱 공유하기" 메뉴
- 카카오톡, 문자, 기타 공유 채널 지원
- 공유 문구: "육아용품 최저가 추적 앱 '아이고'를 써보세요! 내 아이 것은 내가 고른다 🍼" + 스토어 링크

## 계정 연동 플로우
- 온보딩 완료 → 홈 화면 상단에 계정 연동 배너 표시
  - "구글 또는 카카오 계정을 연동하면 데이터가 영구 보존됩니다"
  - [구글 연동] [카카오 연동] [나중에] 버튼
  - "나중에" 선택 시 배너 숨김 (설정에서 언제든 연동 가능)
- 설정 화면 → "계정 연동" 섹션
  - 미연동: [구글 연동] [카카오 연동] 버튼 + 안내 문구
  - 연동 완료: 연동된 계정 표시 (이메일/카카오 닉네임)
- 연동 시: 익명 계정 → 구글/카카오 계정으로 머지 (데이터 유지)
- 연동 강제 없음, 완전 선택사항
