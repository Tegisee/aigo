## 5단계: 로컬 빌드 환경

### 사전 요구사항
- [ ] Xcode 최신 버전 + Command Line Tools 설치
- [ ] fastlane 설치: `brew install fastlane`
- [ ] EAS CLI 최신: `npm install -g eas-cli`

### .easignore 파일
아이고는 GoogleService-Info.plist가 .gitignore에 없으므로 지금이야처럼 .easignore가 필수는 아님.
단, 빌드 아티팩트 제외를 위해 생성 권장:
```
# 지금이야 .easignore 참고 구조
# (빌드 불필요 파일 제외)
builds/
*.ipa
*.apk
*.aab
inspect-output/
```

### 빌드 명령어
```bash
# Production (App Store 제출)
cd ~/aigo/aigo
eas build --local --profile production --platform ios

# Development (실기기 테스트)
eas build --profile development-device --platform ios

# 시뮬레이터
eas build --profile development --platform ios
```

### 주의사항 (지금이야에서 학습)
1. **autoIncrement 함정**: 실패한 빌드도 buildNumber가 증가함
   - 빌드 전 반드시 확인: `eas build:version:get --platform ios`
2. **ios/ 디렉토리**: 이미 존재하면 prebuild 스킵됨 → 네이티브 설정 변경 시 삭제 후 재빌드
3. **빌드 결과물**: app-store 서명된 IPA 파일
4. **IPA 저장**: `~/aigo/builds/ios/aigo-v{version}-{buildNumber}.ipa`

## 6단계: App Store 스크린샷

### 필수 사이즈
| 디바이스 | 해상도 | 필수 | 비고 |
|----------|--------|------|------|
| 6.9" (iPhone 16 Pro Max) | 1320x2868 | ✅ 필수 | 최신 규격 |
| 6.5" (iPhone Xs Max / 14 Plus) | 1242x2688 | ✅ 필수 | 레거시 규격 |
| iPad Pro 12.9" | 2048x2732 | 선택 | supportsTablet: true 시 권장 |

### 스크린샷 촬영 방법 (Xcode 시뮬레이터)

#### 시뮬레이터 준비
```bash
# 사용 가능한 시뮬레이터 목록
xcrun simctl list devices available

# 특정 시뮬레이터 부팅
xcrun simctl boot "iPhone 16 Pro Max"    # 6.9"용
xcrun simctl boot "iPhone 14 Plus"        # 6.5"용

# 시뮬레이터 앱 열기
open -a Simulator
```

#### Preview 빌드로 촬영
```bash
# 시뮬레이터용 빌드 (지금이야는 preview 프로필 사용)
eas build --profile preview --platform ios

# 빌드 후 시뮬레이터에 설치
eas build:run --platform ios --latest
```

#### 스크린샷 캡처
```bash
# 시뮬레이터 스크린샷 (터미널)
xcrun simctl io booted screenshot ~/aigo/screenshots/home_69inch.png

# 또는 시뮬레이터에서 Cmd+S로 직접 캡처
```

### 스크린샷 구성 (권장 9장)
| 순서 | 화면 | 포인트 |
|------|------|--------|
| 1 | 홈 화면 | 아이 닉네임 + 월령 + D-Day + 추천상품 |
| 2 | 카테고리 필터 | 월령별 동적 카테고리 |
| 3 | 상품 상세 | 가격 변동 그래프 + 목표가 |
| 4 | 상품 추가 (공유) | 쿠팡에서 공유 → 자동 등록 |

| 6 | 육아정보 탭 | 예방접종 + 건강검진 |
| 7 | 기념일/시즌 | D-Day + 시즌 컬렉션 |
| 8 | 알림 화면 | 가격 하락 푸시 알림 |

### 언어 설정
- App Store Connect → App Information → Localizations
- 기본 언어: **한국어** 설정
- 스크린샷도 한국어 UI로 캡처
- 영어/일본어 추가 시: 별도 Localization 추가 후 해당 언어 스크린샷 업로드

### 지금이야에서 겪은 문제 & 주의사항
1. **6.9" 스크린샷 해상도 불일치**: Xcode 시뮬레이터 iPhone 16 Pro Max로 재캡처 필요했음
   - 반드시 `xcrun simctl` 사용하여 정확한 해상도 확인
2. **App Store Connect 업로드 오류**: Safari에서 간헐적 업로드 실패 → Chrome 시도 또는 재시도
3. **대한민국 단일 국가 설정**: 처음에 전세계로 설정 후 변경한 이력 — 처음부터 단일 국가로 설정 권장
4. **6.5" 스크린샷**: iPhone Xs Max 또는 iPhone 14 Plus 시뮬레이터 사용
   - 해상도: 1242x2688 (6.5") 정확히 맞아야 업로드 가능
