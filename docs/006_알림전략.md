# 006. 알림 전략

## 원칙
- 귀찮지 않게, 하지만 "우리 아이 것" 이라서 바로 확인하고 싶게
- 육아맘은 알림 피로도가 특히 높음 → 핵심만
- 알림 문구에 아이 감성 반영

## 알림 종류

### 1. 가격 알림 (지금이야 동일)
- 가격 하락 시 알림 발송
- 목표가 도달 시 강조 알림
- 가격 상승/무변동 시 알림 없음

### 2. 재구매 알림 (AIGO 전용)
- 소모품 카테고리만 해당
- 설정한 주기 도래 시 알림
- 현재 최저가 정보 포함

### 3. 아이 기념일 알림
- D-7, D-3, D-1에 푸시 알림 + 홈 배너
- 기념일 구간:
  - 100일 (백일), 200일, 300일
  - 365일 (첫돌)
  - 돌 이후: 매 500일 (500, 1000, 1500, 2000, 2500, 3000…)
  - 특별 숫자: 1111일, 2222일, 3333일, 4444일
  - 매년 생일
- 기념일 도래 시 아이 나이 기반 선물 큐레이션 연동

### 4. 시즌 컬렉션 알림
- 적용 대상: 아이 나이 3세 이상
- 시즌별 노출 시작일에 1회 푸시
  | 시즌 | 날짜 | 푸시 시점 |
  |------|------|----------|
  | 어린이날 | 5/5 | D-30, D-7 |
  | 어버이날 | 5/8 | D-7 |
  | 부부의 날 | 5/21 | D-7 |
  | 크리스마스 | 12/25 | D-30, D-7 |
  | 설날 | 음력 1/1 | D-14 |
  | 추석 | 음력 8/15 | D-14 |
- 나이 기반 추천 카테고리: 장난감, 학습교구, 의류, 전자기기 등
- 어버이날/부부의 날: 부모 자축 선물 추천 (뷰티, 건강식품, 의류, 홈케어)

### 5. 부모 자축 선물 알림
- 아이 기념일 알림 시 부모 자축 문구 함께 발송
  - 예: "[아이고] 1000일 동안 수고하셨어요! 오늘은 나를 위한 선물도 어떨까요? 🎁"
- 어버이날·부부의 날 전용 푸시
  - "[아이고] 어버이날 D-7! 열심히 달려온 나에게 선물 어떨까요? 💐"
  - "[아이고] 부부의 날이 다가와요! 서로에게 작은 선물 어떠세요? 💕"

### 6. 일일 요약 (Phase 2 검토)
- 매일 21시 전후 1회
- "오늘 추적 중인 상품 중 3개가 가격 하락했어요"

## 알림 설정
- 전체 ON/OFF 토글
- 재구매 알림 ON/OFF 토글 (별도)
- 세분화는 Phase 3에서 검토

## 푸시 알림 문구

### 가격 하락
- "[아이고] 찜해둔 기저귀, 가격 내려갔어요! 지금 확인해보세요"
- "[아이고] 우리 아이 분유가 목표가 밑으로! 지금이 기회예요"
- "[아이고] 아이고~ 가격 떨어졌다! 얼른 확인해보세요"

### 목표가 도달
- "[아이고] 드디어! {상품명} 목표가 달성! 지금 구매하러 가볼까요?"
- "[아이고] 기다렸던 그 가격! {상품명} 최저가 도달!"

### 재구매 알림
- "[아이고] 기저귀 재구매할 때가 됐어요. 현재 최저가 {가격}원"
- "[아이고] 분유 떨어질 때쯤이죠? 지금 {가격}원이에요"
- "[아이고] 물티슈 주문할 타이밍! 최저가 확인하기"

### 기념일 알림
- "[아이고] {별명}가 태어난 지 100일이에요! 🎉 백일 선물로 이런 건 어떨까요?"
- "[아이고] {별명}가 1000일을 맞았어요! 🎂 특별한 선물 찾아드릴게요"
- "[아이고] {별명} 생일 D-7! 🎁 올해는 어떤 선물이 좋을까요?"
- "[아이고] {별명}가 태어난 지 1111일! ✨ 특별한 날 특별한 선물"

### 시즌 알림
- "[아이고] 어린이날 D-30! {별명}에게 딱 맞는 선물 찾아볼까요? 🎠"
- "[아이고] 크리스마스 선물 준비하셨나요? {나이}세 인기 선물 모아봤어요 🎄"
- "[아이고] 설날 선물 추천! {별명}에게 어울리는 선물 TOP 10 🧧"

## 재구매 기본 주기
| 카테고리 | 기본 주기 | 조정 범위 | 비고 |
|---------|----------|----------|------|
| 기저귀 | 14일 | 7~30일 | 소모품 |
| 분유 | 21일 | 14~45일 | 소모품 |
| 물티슈 | 30일 | 14~60일 | 소모품 |
| 기타 소모품 | 30일 | 7~90일 | 소모품 |
| 내구재 (장난감 등) | — | — | 알림 없음 |

## 알림 계산 로직

### 월령 자동 계산
- 아이 생년월일 → 현재 월령 매일 자동 계산
- 월령 변경 시(구간 진입) → 푸시 알림
  - 예: "우리 아이 7개월이 됐어요! 이유식 준비해볼까요?"
  - 구간: 신생아(0~6) → 이유식기(7~12) → 걸음마기(13~24) → 유아기(25~48) → 미취학(49~84) → 초등(85~156)

### 재구매 알림 계산
- 기준일: `lastPurchasedAt` (구매일) 우선, 없으면 `createdAt` (등록일) fallback
- 기준일 + `repurchaseDays` = 다음 재구매 예정일
- D-3부터 알림 표시, D-day에 푸시 발송
- "구매했어요" 버튼 탭 → `lastPurchasedAt` 갱신 → 주기 리셋

### 복합 알림 (가격 하락 + 재구매 임박)
- 가격 하락과 재구매 임박이 동시 충족 시 → 우선순위 높은 복합 알림 발송
- 예: "기저귀 살 때가 됐는데, 지금 역대 최저가예요!"
- 복합 알림 문구:
  - "[아이고] 타이밍 완벽! {상품명} 재구매 시기인데 가격도 떨어졌어요"
  - "[아이고] 지금 안 사면 후회할걸요! {카테고리} 최저가 + 재구매 시기"
