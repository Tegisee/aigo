# 004. 수익모델

## 전략: 쿠팡 파트너스 단일 모델
- 수수료율: 카테고리별 3~10%
- 계정: 지금이야와 동일 파트너스 계정 사용 가능
- 모든 "구매하기" 링크 → 파트너스 딥링크 경유

## 육아용품 특화 강점
- 기저귀/분유/물티슈 = 소모품 반복 구매 → 높은 전환율
- 재구매 알림 → 파트너스 딥링크 → 자연스러운 수익 발생
- 평균 객단가: 기저귀 3~5만, 분유 3~4만, 카시트/유모차 20~80만

## 수익 발생 경로 (subId 트래킹)
| 경로 | subId | 설명 |
|------|-------|------|
| 추적 상품 구매 | `tracked` | 상세화면 "지금 구매하기" |
| 골드박스 클릭 | `goldbox` | 홈 상단 오늘의 특가 |
| 상품 공유 | `share` | 타 사용자에게 공유 시 |
| 쿠팡 홈 이동 | `home` | "추적상품 가져오기" 버튼 |

## 수익 극대화 전략
1. **재구매 알림 → 구매 유도**: 소모품 주기 도래 시 알림 + 최저가 표시 + 원터치 구매
2. **골드박스 노출**: 홈 상단 특가 → 충동 구매 유도 (제휴 수수료 발생)
3. **목표가 달성 알림**: 기다렸던 가격 → 즉시 구매 전환율 극대화
4. **공유 기능**: 공유된 링크도 제휴 링크 → 바이럴 수익

## 파트너스 필수 안내 문구
- 홈, 상세, 설정 화면 하단에 고정 노출
- "이 앱은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."
