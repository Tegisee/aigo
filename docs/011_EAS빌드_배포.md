## 실기기 테스트 방법
1. EAS Build로 Development Build 1회 생성
2. 실기기에 설치
3. 맥미니에서 `npx expo start` 실행
4. 실기기와 같은 WiFi → 자동 연결
5. 코드 수정 → 저장 → 실기기 즉시 반영 (Fast Refresh)

## 빌드 명령어
- Android: `eas build --profile development --platform android`
- iOS 실기기: `eas build --profile development-device --platform ios`
- iOS 시뮬레이터: `eas build --profile development --platform ios`
- 실행: `eas build:run --platform android --latest`
