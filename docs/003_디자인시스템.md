# 003. 디자인 시스템

## 컬러
- background: #FFF8F0 (웜 크림)
- card: #FFFFFF
- primary: #FF7E67 (코랄 핑크 — 따뜻함, 육아 감성)
- secondary: #4ECDC4 (민트 — 포인트, 할인/절약 강조)
- text: #2D3436 (소프트 블랙)
- subtext: #999999
- border: #F0E6D8 (웜 베이지)
- danger: #FF6B6B (삭제/경고)
- success: #00B894 (목표가 달성)

## 컨셉
- 따뜻한 코랄 + 크림 조합 — 육아맘 감성
- 지금이야(민트+블랙)와 완전히 다른 톤 → 자매앱이지만 독립 정체성
- 라이트모드 기본 (다크모드 Phase 3에서 검토)
- 둥글고 부드러운 UI 요소 (borderRadius 16~20)
- 아이콘: 귀여운 라인 아이콘 (Ionicons 유지)

## 지금이야 vs AIGO 비교
| 요소 | 지금이야 | AIGO |
|------|---------|------|
| 톤 | 다크 + 쿨 | 라이트 + 웜 |
| 메인 컬러 | #00E5CC 민트 | #FF7E67 코랄 |
| 배경 | #000000 블랙 | #FFF8F0 크림 |
| 감성 | 테크/유틸리티 | 따뜻함/육아 |
| 모드 | 다크 전용 | 라이트 기본 |

## 화면 목록
1. **메인 리스트 (홈 탭)**
   - 헤더: "아이고" + 아이 나이 배지 (예: "24개월")
   - 카테고리 필터 칩 (전체/기저귀/분유/물티슈/의류/장난감/가구)
   - 상품 카드 FlatList
   - FAB(+): URL 붙여넣기 모달
2. **목표가 입력 모달**
   - URL 입력 (Share Intent 자동)
   - 카테고리 자동 분류 + 수동 변경
   - 목표가 숫자 입력 + 추천 목표가
   - 재구매 알림 설정 (소모품일 경우)
   - 저장/취소 버튼
3. **상품 상세 화면**
   - 뒤로가기 + 삭제 버튼
   - 썸네일 + 상품명 + 카테고리 태그
   - 가격 그리드 2x2 (현재가/목표가/최고가/최저가)
   - 30일 가격 변동 차트
   - 재구매 주기 표시 (소모품)
   - "지금 구매하기" 버튼
4. **설정 (설정 탭)**
   - 아이 생년월일 설정
   - 알림 ON/OFF
   - 재구매 알림 주기 기본값 설정
   - 개인정보처리방침
   - 파트너스 안내 문구
   - 버전 정보

## 상품 카드 구성요소
- 카테고리 태그 (코랄 배지)
- 썸네일
- 상품명
- 현재가 / 목표가
- 달성률 % + 프로그레스바 (코랄→민트 그라데이션)
- Sparkline 차트
- 재구매 D-day 표시 (소모품만, 예: "재구매 D-3")
- 탭 시 상세화면 이동

## 앱 아이콘 컨셉
- 코랄 배경 + 쇼핑백 + 아기 발바닥 or 하트
- 둥글고 따뜻한 느낌
- 앱스토어에서 육아 카테고리임을 즉시 인지 가능
