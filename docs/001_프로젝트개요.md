# 001. 프로젝트 개요

## 앱 기본 정보
- 앱 이름: 아이고 (AIGO)
- 번들 ID: com.aigo.app
- 슬로건: "내 아이 것은 내가 고른다"
- 컨셉: 육아용품 전문 쿠팡 최저가 추적 앱. 로그인 없이 공유만 하면 내 아이에게 필요한 것을 가장 싼 타이밍에 알려주는 부모 전용 가격 비서.

## 이름 의미
- 아이고(AIGO) = "아이" + "GO" — 아이를 위해 달려가는 부모의 마음
- 한국어 감탄사 "아이고~" — 가격 떨어졌을 때의 반가움
- 영문 표기 AIGO — 글로벌 확장 고려

## 왜 육아용품 전문인가?

### 시장 특성
- 기저귀/분유/물티슈: 매달 반복 구매, 가격 민감도 극도로 높음
- 유모차/카시트/젖병: 고가 제품, 세일 타이밍이 곧 수만~수십만원 절약
- 육아맘/대디 커뮤니티: 최저가 정보 공유가 가장 활발한 분야
- 쿠팡 로켓배송/로켓직구: 육아용품 카테고리 가격 변동 빈번

### 기존 가격 추적 앱의 한계
- 범용 앱은 육아용품에 최적화 안 됨
- 아이 나이별 필요 용품이 달라지는데 필터 없음
- 소모품 재구매 주기 알림 없음
- 육아맘 감성에 안 맞는 딱딱한 UI

### AIGO의 답
- 아이 생년월일 입력 → 연령대별 추천 카테고리 자동 세팅
- 소모품 재구매 주기 알림 (기저귀 2주, 분유 3주 등)
- 따뜻하고 직관적인 UI
- 쿠팡 공유 한 번이면 끝 — 지금이야와 동일한 초간단 UX

## 핵심 기능
1. 쿠팡 공유하기 → 육아용품 자동 인식 + 카테고리 분류
2. 목표가 설정 → 가격 하락 시 푸시 알림
3. 아이 나이별 카테고리 필터 (신생아~초등학생)
4. 소모품 재구매 주기 알림
5. 가격 변동 그래프
6. 월령별 인기 상품 추천 (아이고 사용자 데이터 기반)
7. 아이 기념일 알림 + 선물 큐레이션 (백일/돌/1000일/생일)
8. 시즌 컬렉션 추천 (어린이날/크리스마스/설날/추석)
9. "사주세요" 공유 — 가족/지인에게 상품 링크 공유 (텐 포켓 활용)
10. 앱 공유하기 — 다운로드 링크 카카오톡/문자 공유

## 홈 화면 구조
1. **상단**: 아이 별명 + 월령 + 태어난 지 N일째 + 계정 연동 배너
   - 예: "쪼꼬미 · 8개월 · 태어난 지 245일째 🌱"
2. **기념일 배너** (D-7부터 자동 노출): "쪼꼬미가 태어난 지 100일이에요! 🎉"
3. **시즌 컬렉션** (해당 시즌 D-day 기간에만 노출, 3세 이상)
4. **섹션1 — "○개월 맘들의 픽"**: trackerCount 기반 인기 상품 (가로 스크롤)
5. **섹션2 — "이번 주 핫딜"**: 최근 가격 하락률 높은 상품 (가로 스크롤)
6. **섹션3 — "내가 추적 중인 상품"**: 등록한 상품 목록
7. **섹션4 — "재구매 알림 임박"**: D-day 배지 상품

## 아이 프로필
- 별명: 홈 화면·알림 문구에 사용 (예: "쪼꼬미")
- 생년월일: 월령 자동 계산, 기념일, 구간별 큐레이션
- 성별: 남아 / 여아 / 선택안함
- 모든 항목 선택사항, 설정에서 언제든 수정 가능

## 성별 기반 큐레이션
| 성별 | 우선 추천 카테고리 |
|------|------------------|
| 남아 | 자동차, 로봇, 공룡, 블록, 스포츠용품 |
| 여아 | 인형, 역할놀이, 미술/공예, 액세서리 |
| 선택안함 | 성별 무관 인기 상품 |

- 기념일·시즌 선물 추천에도 성별 반영
  - 예: "쪼꼬미(남아) 어린이날 D-7! 8세 남아 인기 선물 TOP 10"
  - 예: "쪼꼬미(여아) 생일 D-3! 6세 여아가 좋아하는 선물 추천"
- 쿠팡 파트너스 키워드 검색 시 성별 키워드 포함하여 정확도 향상

## 월령별 인기 상품 추천
- shared_products의 trackerCount + ageGroup 데이터 활용
- 사용자 아이 월령 + 성별에 맞는 구간 자동 매칭하여 표시
- 정렬 기준:
  1. trackerCount 높은 순 (많이 추적 중인 상품)
  2. 가격 하락률 높은 순 (핫딜)
  3. 최근 가격 하락 상품 (최신순)
- 초기(사용자 적을 때): 쿠팡 베스트셀러 기반 표시
- 성장 후: 아이고 실제 사용자 데이터 기반으로 자동 전환
- 섹션명 예시: "4개월 아기 맘들의 픽", "이번 주 핫딜"

## "사주세요" 공유 기능
- 상품 상세 화면에서 "사주세요" 버튼 → 파트너스 딥링크 생성 → 카카오톡/문자/기타 공유
- 받는 사람이 링크 클릭 후 24시간 내 구매 시 파트너스 수익 발생
- subId: `share_buyme`로 트래킹
- 텐 포켓 문화 활용: 아이 1명을 위해 가족·지인 10명이 지갑을 열어주는 문화
- 공유 문구 예시:
  - "쪼꼬미가 이거 갖고 싶대요! 눌러보세요 👇"
  - "할머니~ 이거 사주세요! 💝"
  - "아빠~ 쪼꼬미 생일선물이에요!"

## 앱 공유하기
- 홈 화면 헤더 공유 버튼 / 설정 화면에서 앱 다운로드 링크 공유
- 카카오톡, 문자, 기타 공유 채널 지원
- 공유 문구: "육아용품 최저가 추적 앱 '아이고'를 써보세요! 내 아이 것은 내가 고른다 🍼"

## 재구매 알림 구조
- 쿠팡 앱과 데이터 연동 불가 → 사용자가 직접 "구매했어요" 버튼으로 구매일 기록
- 구매일 미기록 시 → 추적 등록일 기준으로 주기 계산 (fallback)
- 예: "기저귀 등록한 지 14일 됐어요. 슬슬 필요할 때 아닌가요?"
- "구매했어요" 탭 시 → 구매일 갱신 → 주기 리셋

## 아이 기념일 알림 + 선물 큐레이션
- 기념일 D-7, D-3, D-1 전부터 홈 화면 배너 노출
- 기념일 도래 시 아이 나이 기반 맞춤 선물 큐레이션 (쿠팡 파트너스 키워드 검색)
- 기념일 구간:
  - 100일 (백일)
  - 200일, 300일
  - 365일 (첫돌)
  - 돌 이후: 매 500일 단위 (500, 1000, 1500, 2000, 2500, 3000…)
  - 특별 숫자: 1111일, 2222일, 3333일, 4444일
  - 매년 생일 자동 알림
- 알림 문구 예시:
  - "쪼꼬미가 태어난 지 100일이에요! 🎉 백일 선물로 이런 건 어떨까요?"
  - "쪼꼬미가 1000일을 맞았어요! 🎂 특별한 선물 찾아드릴게요"
  - "쪼꼬미 생일 D-7! 🎁 올해는 어떤 선물이 좋을까요?"

## 부모 자축 선물
- 아이 기념일 (100일, 500일, 1000일, 생일 등) 알림 시 부모를 위한 자축 선물 섹션 함께 노출
  - 예: "1000일 동안 수고하셨어요! 오늘은 나를 위한 선물도 어떨까요?"
- 부모 전용 시즌:
  | 시즌 | 날짜 | 노출 시작 |
  |------|------|----------|
  | 어버이날 | 5월 8일 | D-7 |
  | 부부의 날 | 5월 21일 | D-7 |
- 추천 카테고리: 뷰티, 건강식품, 의류, 홈케어
- 쿠팡 파트너스 키워드 검색 기반 큐레이션 (파트너스 수수료 발생)

## 시즌 컬렉션
- 적용 대상: 아이 나이 3세 이상
- 홈 화면에 D-day 기준으로 자동 섹션 등장
- 시즌 목록:
  | 시즌 | 날짜 | 노출 시작 | 추천 카테고리 |
  |------|------|----------|--------------|
  | 어린이날 | 5월 5일 | D-30 | 장난감, 학습교구, 의류 |
  | 크리스마스 | 12월 25일 | D-30 | 장난감, 전자기기, 의류 |
  | 설날 | 음력 1월 1일 | D-14 | 의류, 도서, 완구 |
  | 추석 | 음력 8월 15일 | D-14 | 의류, 식품, 완구 |
- 아이 나이 기반 맞춤 추천 (예: 8세 → "8세 어린이날 인기 선물 TOP 10")
- 쿠팡 파트너스 키워드 검색으로 시즌별 상품 자동 큐레이션

## 로그인 정책
- 기본: Firebase Anonymous Auth (로그인 없이 즉시 사용)
- 선택 연동: 구글 로그인 / 카카오 로그인
- 익명 → 구글/카카오 계정 머지(merge) 지원
- 미연동 시 안내: "앱 삭제 또는 기기 변경 시 데이터가 초기화됩니다. 구글 또는 카카오 계정 연동 시 데이터가 영구 보존됩니다."
- 카카오 로그인: Kakao SDK + Firebase Custom Token 방식 (Firebase Functions 필요)
- 구글 로그인: Firebase 기본 지원
- 연동 강제 없음, 완전 선택사항

## 절대 원칙 (지금이야 동일 + 추가)
1. 로그인 강제 없음 (익명 즉시 사용, 계정 연동은 선택)
2. 타 판매자 검색 로직 절대 금지
3. 커뮤니티/포인트/소셜 기능 없음
4. 직접 크롤링 금지 (쿠팡 파트너스 API만 사용)
5. 광고(AdMob) 없음
6. 문의하기 없음
7. 상품 등록 최대 30개 제한 (육아 소모품 특성상 지금이야 20개보다 여유)
8. **의료/건강 조언 절대 금지** — 가격 추적만
9. **아이 개인정보 최소 수집** — 생년월일만 (이름 X, 사진 X)

- 주 타겟: 육아맘/육아대디
- 기저귀/분유 가격을 매주 체크하는 가격 민감 부모

- 둘째/셋째 임신 중인 부모 (재구매 패턴 명확)
- 초등 학부모 (학용품/교구/전자기기 가격 추적)

## 개발 일정 (예상)

- Phase 2 (알림 + 특화): 재구매 알림 + 나이별 필터 (~1주)
- Phase 3 (성장): 스토어 출시 + 마케팅
